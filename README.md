# рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ
crocobet-laravel
## ­ЪЊќ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

рЃћрЃА рЃљрЃарЃўрЃА RESTful API рЃърЃарЃЮрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ Laravel-рЃќрЃћ рЃЊрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃўрЃА, рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА. рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃћрЃЉрЃА, рЃарЃљрЃЌрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃА API-рЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ.

## ­Ъџђ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃћрЃЉрЃў

**рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ**:
  - рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ.
  - рЃбрЃЮрЃЎрЃћрЃюрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ Laravel-рЃўрЃА guard рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃЌ.
**рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ**:
  - рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ, рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ.
  - policy рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃгрЃљрЃерЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
**Middleware рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ**:
  - рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃўрЃА Middleware рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃарЃћрЃЦрЃЋрЃћрЃАрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃџрЃЮрЃњрЃљрЃЊ.
**рЃарЃЮрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў**:
  - рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ Gates рЃЊрЃљ Policies рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.
**Unit рЃЊрЃљ Feature рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ**:
  - рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ.


**рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў**
- PHP 8.1 рЃљрЃю рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ

# API рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

## ­ЪјЪ№ИЈ рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

### рЃбрЃЮрЃЎрЃћрЃюрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- **рЃЏрЃћрЃЌрЃЮрЃЊрЃў:** `POST`
